# 関数説明書

## ファイル概要
**ファイル名:** utils.py  
**説明:** ユーティリティ関数集

---

## 関数一覧

### 1. calculate_area(width, height)

**説明:** 長方形の面積を計算する

**パラメータ:**
- `width`: 長方形の幅
- `height`: 長方形の高さ

**処理内容:**
- 幅と高さを掛け算して面積を計算

**戻り値:**
- 長方形の面積（数値）

**使用例:**
```python
area = calculate_area(10, 20)  # 結果: 200
```

---

### 2. format_price(price)

**説明:** 価格を通貨形式にフォーマット

**パラメータ:**
- `price`: フォーマットする価格（数値）

**処理内容:**
- 3桁ごとにカンマを入れて円記号を付ける

**戻り値:**
- フォーマットされた価格文字列（例: "¥1,000"）

**使用例:**
```python
formatted = format_price(98000)  # 結果: "¥98,000"
```

---

## メイン処理

ファイルを直接実行した場合、以下のテスト実行が行われます：
- `calculate_area(10, 20)` を実行し、結果（200）を出力
